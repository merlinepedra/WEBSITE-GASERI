    Broj $\pi$ mo≈æe se aproksimirati formulom

    $$ \pi = \int_0^1 \frac{4}{1 + x^2} dx \approx \frac{1}{n} \sum_{i = 0}^{n - 1} \frac{4}{1 + (\frac{i + 0.5}{n})^2}. $$

    • sekvencijalni kod je oblika

      import numpy as np

n_iterations = 10

def compute_pi(n):
    h = 1.0 / n
    s = 0.0
    for i in range(n):
        x = h * (i + 0.5)
        s += 4.0 / (1.0 + x**2)
    return s * h

pi = compute_pi(n_iterations)
error = abs(pi - np.math.pi)
print("pi is approximately %.16f, error is approximately %.16f" % (pi, error))

    Aproksimativno raƒçunanje broja $\pi$ kori≈°tenjem Monte Carlo metode

    Monte Carlo metode su skupina metoda koja na temelju sluƒçajnih brojeva i velikog broja pokusa daju aproksimaciju odreƒëene vrijednosti.

    Broj $\pi$ mo≈æe se aproksimativno raƒçunati i kori≈°tenjem Monte Carlo metode. Da bi to vidjeli, uzmimo da je u koordinatnom sustavu ucrtan jediniƒçni krug unutar jediniƒçnog kvadrata. Promotrimo njegovu ƒçetvrtinu u prvom kvadrantu:

    • ƒçetvrtina kvadrata je povr≈°ine 1 (obzirom da su mu obje stranice duljine 1),
    • ƒçevrtina kruga je povr≈°ine $\frac{\pi}{4}$ (obzirom da mu je radijus 1).

    Sluƒçajno odabrana toƒçka unutar ƒçetvrtine kvadrata ima vjerojatnost $\frac{\pi}{4}$ da upadne unutar ƒçetvrtine kruga. Dakle, ako s $n$ oznaƒçimo broj sluƒçajno odabranih toƒçaka, a s $h$ broj toƒçaka koje se od $n$ sluƒçajno odabranih nalaze unutar ƒçetvrtine kruga, aproksimativno mo≈æemo odrediti broj pi kao

    $$ \pi = 4 \times \frac{h}{n}. $$

    Poveƒáavajuƒái $n$ dobivamo toƒçniju aproksimaciju, a navedena metoda naziva se Monte Carlo metoda. Programski kod koji aproksimira broj pi kori≈°tenjem Monte Carlo metode je oblika:

    • sekvencijalni kod

      import numpy as np

n_random_choices = 10000
hits = 0
throws = 0

for i in range (0, n_random_choices):
    throws += 1
    x = np.random.random()
    y = np.random.random()
    dist = np.math.sqrt(x * x + y * y)
    if dist <= 1.0:
        hits += 1

pi = 4 * (hits / throws)

error = abs(pi - np.math.pi)
print("pi is approximately %.16f, error is approximately %.16f" % (pi, error))
