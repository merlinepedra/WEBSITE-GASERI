    Kori≈°tenje izgradnje kontejnera u kompoziciji

    U kompoziciji je moguƒáe izgraditi neke od kontejnera na temelju datoteka Dockerfile. Za ilustraciju mo≈æemo iskoristiti primjer dan u Get started with Docker Compose, web aplikaciju napisanu u Pythonu kori≈°tenjem Flaska koja dohvaƒáa podatke iz Redisa. Datoteka app.py je sadr≈æaja:

import redis
from flask import Flask

app = Flask(__name__)
cache = redis.Redis(host='mojredis', port=6379)

def get_hit_count():
    retries = 5
    while True:
        try:
            return cache.incr('hits')
        except redis.exceptions.ConnectionError as exc:
            if retries == 0:
                raise exc
            retries -= 1
            time.sleep(0.5)

@app.route('/')
def hello():
    count = get_hit_count()
    return 'Hello World! I have been seen {} times.\n'.format(count)

    Jasno je vidljivo iz naredbi import da ova aplikacija za svoj rad zahtijeva Python module flask i redis (podsjetimo se da je modul time dio standardne biblioteke). Stvorimo datoteku requirements.txt koju ƒáe pip koristiti za preuzimanje potrebnih modula:
